# Rough idea — Hoopoe cockpit

A desktop app for the Agentic Coding Flywheel. Sits on top of the existing CLI-driven workflow
(NTM tmux swarms, `br`/`bv` beads, Agent Mail, build helpers, code health) and gives one
operator console. Don't reimplement the engine; just visualize, automate, and audit it.

Open questions:

- macOS only for v1, or cross-platform?
- VPS-only execution, or local fallback?
- BYOK or subscription-only for model access?
